# imitativeJD
imitative jd.com(Wechat mini apps) use uniapp+vuejs

该项目使用了uniapp和vuejs技术，为模仿[京东商城](https://jd.com)的微信小程序版本。

## Technology Stack（技术栈）:

1. uniapp(cross platform tools)
2. vuejs2.x(js framework)
3. uView1.3.2(UI components)
